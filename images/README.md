Create the following images to complete SEO setup:

- **Used on**: Homepage, all social shares

### blog-og.jpg

## Tips for OG Images

1. **Text should be large** - Readable at thumbnail size
2. **Use brand colors** - Teal (#0d9488) and amber (#d97706)
3. **Include logo** - Top left or centered
4. **Keep it simple** - Less is more at small sizes
5. **Test on social media** - Use Facebook Sharing Debugger
