Technology Stack

Supabase
PostgreSQL database, authentication, storage, and edge functions in one platform.
n8n
Self-hosted workflow automation for SEO, social media, and operational tasks.
Claude Code
AI agent for SEO tasks, content generation, and automated optimization.
Stripe Payments
Standard payment processing for botanical extracts (2.9% + 30¢).
